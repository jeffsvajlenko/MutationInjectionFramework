AIST CCFinderX ver. 10.2.7.4 for WinXP
======================================

( C ) 2009-2010 AIST.

http://www.ccfinder.net/

System requirements
-------------------

Windows XP or later (32bit), JRE ver. 1.5 or later, Python 2.6 or later.
(However, CCFinderX has been tested on Windows 7, JRE 1.6-u19, Python 2.6.5.)

Install
-------

(1) Install the following software products.

  - JRE (Java Runtime) for Windows 32bit ver. 1.5 or later
  - Python 32bit ver. 2.6 or later (not 3.x)

(2) Make a directory for CCFinderX (like: c:\ccfx) and expand ccfx-win32-en.zip 
    in the directory.

(3) Run /bin/gemx.bat of the directory.

Uninstall
---------

To uninstall, remove the installation directory.
(CCFinderX doesn't touch Windows registory)
